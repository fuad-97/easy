# Frontend

واجهة Next.js عربية RTL Mobile-first للتاجر والعميل.

## التشغيل

```bash
copy .env.example .env.local
npm install
npm run dev
```

## الصفحات المنفذة

- `/`
- `/merchant/register`
- `/merchant/login`
- `/merchant/onboarding`
- `/merchant/dashboard`
- `/merchant/products`
- `/merchant/orders`
- `/merchant/customers`
- `/merchant/settings`
- `/s/[slug]`
- `/s/[slug]/products/[id]`
- `/s/[slug]/checkout`

## البيئة

- `NEXT_PUBLIC_API_URL=https://your-api-domain.up.railway.app`
