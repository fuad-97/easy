# نصوص UX مقترحة

## شاشة البداية
- ابدأ متجرك بسهولة
- أنشئ متجرك خلال دقائق واستقبل طلباتك من مكان واحد
- إنشاء متجر جديد
- تسجيل الدخول

## لوحة التحكم
- عندك {n} طلبات جديدة
- طلبان موعد تسليمهما اليوم
- أكثر منتج طلبًا هذا الأسبوع
- إضافة منتج
- مشاركة المتجر
- تعديل أوقات العمل
- إنشاء عرض

## الطلبات
- جديد
- تم التأكيد
- قيد التحضير
- جاهز
- خرج للتوصيل
- تم التسليم
- ملغي

## نجاح الطلب
- تم استلام طلبك بنجاح
- رقم الطلب: {order_number}
- سيتم التواصل معك قريبًا
- متابعة عبر واتساب
- الرجوع للمتجر
