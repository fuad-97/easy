# UI Direction

## Principles
- Arabic-first, RTL-first
- Mobile-first
- Warm, clean, modern
- Large touch targets
- Soft cards and rounded corners
- Big product imagery

## Suggested Fonts
- Arabic: Cairo or IBM Plex Sans Arabic
- English: Inter

## Suggested Components
- Bottom navigation for merchant app
- Card-based dashboard
- Simple tabs for order states
- Large CTA buttons
- Clear empty states

## Visual Tone
- Friendly
- Trustworthy
- Lightweight
- Not corporate-heavy
